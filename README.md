Flask Calculator 🧮

A simple calculator web application built with Python Flask, HTML, and CSS.
This project allows users to perform basic arithmetic operations through a simple web interface.

🚀 Features

Addition

Subtraction

Multiplication

Division

Simple and clean user interface

Built using Flask backend

▶️ How to Run the Project

Clone the repository

git clone https://github.com/your-username/flask-calculator.git

Go to the project folder

cd flask-calculator

Install Flask

pip install flask

Run the application

python app.py

Open your browser and visit

http://127.0.0.1:5000
